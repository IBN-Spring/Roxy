# Roxy Agent v1.0.1

Patch release for the v1.0 stable source-evolving agent release.

## Highlights

- Bumped package version to `1.0.1`.
- Fixed `roxy dev check --quick` so version validation compares `roxy.__version__` with `pyproject.toml` instead of a hardcoded old version.
- Hardened Windows subprocess output handling in dev checks with UTF-8 decoding and replacement for invalid bytes.
- Updated README badges and Quick Start paths for the `IBN-Spring/Roxy-Agent` repository.
- Removed stale mascot image from the English README.

## Validation

```bash
python -m roxy --version
# roxy 1.0.1

python -m roxy dev check --quick
# 8/8 checks passed

python -m pytest tests
# 308 passed
```

## Notes

This release does not add new runtime features. It fixes release-readiness checks and documentation polish after `v1.0.0`.
